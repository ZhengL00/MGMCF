# MGMCF

This repository contains the source code for our paper. 

## Setup
- **Build environment**
```
cd MGMCF
# use anaconda to build environment 
conda create -n MGMCF python=3.10
conda activate MGMCF
# install packages
pip install transformers==4.11.3
pip install numpy==1.23.3
```

## Quick Start

```
python main.py
```
